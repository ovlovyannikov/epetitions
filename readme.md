## Електронні петиції

Сайт разработан с помощью PHP frameworka - Laravel 5.1

Требования:
"php": ">=5.5.9"
"mysql": ">=5.5.34"

1. Создайте новую базу данных:
petition
2. Создайте пользователя:
petition с паролем (123456)
3. Выполните миграцию таблиц выполняются командой:
php artisan migrate
4. Питиція з’являється на сайті після перевірки (check=1)
